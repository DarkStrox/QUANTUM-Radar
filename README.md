# ⚡ QuRadar - Traffic Monitoring System

A traffic surveillance and violation enforcement system available in both Java and [Python](./python%20version).

---

## 🚀 Quick Start

### Java Version
1. **Compile Java Files**:
   ```bash
   javac *.java
   ```
2. **Run Demonstration**:
   ```bash
   java Main
   ```

### Python Version
1. **Navigate to Python directory**:
   ```bash
   cd "python version"
   ```
2. **Run Demonstration**:
   ```bash
   python main.py
   ```
